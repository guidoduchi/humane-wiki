# Page Template

Use this template when creating new HUMANE wiki pages.

```md
# Page Title

Status: Draft
Spoiler Level: Full Canon

## One-Sentence Purpose

What does this page help us understand or decide?

## Canon Summary

What is true right now?

## Narrative Function

Why does this matter to the story?

## Reader Experience

What should the reader feel, understand, suspect, or misunderstand?

## Hidden Canon

What can the wiki know that the reader should not know yet?

## Continuity Notes

- Timeline:
- Location:
- Related characters:
- Related systems:

## Open Questions

- What still needs a decision?
```

## Template Rule

Not every page needs every section, but every important page should distinguish between:

- fact,
- function,
- reader experience,
- hidden canon,
- open questions.
