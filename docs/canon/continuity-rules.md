# Continuity Rules

Status: Draft
Spoiler Level: Full Canon

Continuity rules keep the wiki useful instead of decorative.

## Authority Order

1. Canon Source of Truth
2. Current character/system/world pages
3. Current book continuity pages
4. Working notes
5. Old drafts and deprecated material

## Contradiction Rule

If something beautiful contradicts canon, do not delete it automatically.

Choose one:

- revise the detail,
- revise canon deliberately,
- mark it as prototype,
- move it to open questions.

## Promotion Rule

An idea becomes stable only when it has:

- narrative function,
- timeline placement,
- character consequence,
- no unresolved contradiction.
