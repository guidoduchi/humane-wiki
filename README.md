## Repository Boundary

This wiki is for:

- canon rules,
- timeline tracking,
- character and culture profiles,
- fictional systems,
- chapter continuity support,
- project collaboration,
- narrative development decisions.

The actual manuscript prose belongs in the separate manuscript repository.

## Development Rule

Every page should help answer at least one of these questions:

1. What is true right now?
2. Why does it matter to the story?
3. What should the reader feel?
4. What must remain hidden until later?
5. What breaks if this detail changes?
